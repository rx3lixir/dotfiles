
# My Arch linux dotfiles

As simple as that. Made with "Nord" color palette with great neovim config for go and typescript development.


## Features

- Clean
- Minimalistic
- Simple
- All u actually need including waybar, great neovim config, tmux, fish, 
- May be a good start for your own customization
